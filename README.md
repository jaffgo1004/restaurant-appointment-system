# Restaurant Appointment System

A web-based system that allows customers to book restaurant appointments online.

## Team Members and Roles
- Wenxin – Developer (Frontend)
- Fonglam – Developer (Logic & Validation)
- Jaff – Scrum Master
- Chang – Product Owner & Tester

## System Overview
This system allows users to book restaurant appointments online by selecting a stall and time.

## Features
- Online appointment booking
- Stall and time selection
- Appointment display
- Input validation

## How to Run
Open index.html in a web browser.

## Scrum Process Summary
The project was developed using two sprints with product backlog, sprint planning, review, and retrospective.

## Contributions
Each team member contributed to development, documentation, and testing.
