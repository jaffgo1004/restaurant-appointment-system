# Product Backlog

US1: As a customer, I want to book a restaurant appointment online so that I do not need to call the restaurant.

US2: As a customer, I want to select a food stall and booking time so that I can plan my visit.

US3: As staff, I want to view all current appointments.

US4: As a system, I want to validate input to prevent incomplete bookings.

US5: As a user, I want to receive clear confirmation messages.
